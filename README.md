# dotfile
Nothing special, just a backup of my vim configuration.

# Installation
1. Download and extract .vim and .vimrc to home folder.
2. Install Vundle
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
3. open vim and run 
```
:VundleInstall
```
